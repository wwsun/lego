# lego
A koa-based web app for big data visualization

## Introduction

1. node v4
2. koa

## Development

Install dependencies

    npm install

Build server-side codes with Gulp

    gulp
    
Build client-side codes with Webpack

    npm run webpack-hot
    
Open your browser to visit `http://localhost:9001/`
